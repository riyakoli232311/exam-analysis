## ğŸ“„ CSV Format

When uploading test results via CSV, use this column structure:

```csv
subject,topic,is_correct,time_spent_seconds,difficulty
Mathematics,Algebra,true,90,easy
Physics,Mechanics,false,180,hard
Chemistry,Organic,true,60,medium
```

| Column | Type | Description |
|--------|------|-------------|
| `subject` | string | Subject name (e.g. Mathematics) |
| `topic` | string | Specific topic within the subject |
| `is_correct` | boolean | `true` or `false` / `1` or `0` |
| `time_spent_seconds` | number | Time taken to answer in seconds |
| `difficulty` | string | `easy`, `medium`, or `hard` |
